# Quiz Application - DigComp 2.2 Area 1

This is a static web application implementing the DigComp 2.2 Area 1 digital competence framework. It includes an interactive quiz with 180 questions across 3 chapters and difficulty levels, plus a theory section.

## Features

- 180 quiz questions with timer (30 minutes)
- Chapter-based question distribution
- Difficulty levels (Easy, Medium, Hard)
- Dark mode theme with Tailwind CSS
- Responsive design

## How to Use

1. Open `index.html` in a browser
2. Select a chapter and difficulty
3. Answer questions within the time limit
4. View results and restart

## Files

- `index.html` - Main application entry point
- `script.js` - Quiz logic and state management
- `style.css` - Styling and theming
- `teoria.html` - Theory section
- `teoria.js` - Theory navigation

## Deployment

This static site can be deployed to GitHub Pages by pushing to a repository and enabling GitHub Pages from the root branch.
